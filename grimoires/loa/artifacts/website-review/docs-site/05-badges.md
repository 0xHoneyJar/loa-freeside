## Overview

> "Badges are achievement markers that recognize specific community milestones and behaviors." Unlike tiers, badges are cumulative - members can earn multiple simultaneously.

---

## Holder Badges

| Badge | Criteria | Description |
|-------|----------|-------------|
| Diamond Hands | 180+ days held | "Never sold through major dips" |
| OG | Held since first 30 days | "Early community member" |
| Whale | Top 1% by balance | "Major holder" |
| Shrimp | Bottom 25% by balance | "Small but committed" |

---

## Behavior Badges

| Badge | Criteria | Description |
|-------|----------|-------------|
| Accumulator | 3+ buy transactions | "Consistently adding to position" |
| Peak Survivor | Held through 50%+ drawdown | "Weathered major corrections" |
| Governance Active | 5+ on-chain votes | "Active in protocol governance" |
| Sniper | Bought within 1 hour of launch | "Quick to commit" |

---

## Milestone Badges

| Badge | Criteria | Description |
|-------|----------|-------------|
| 1 Year Club | 365 days held | "Full year of holding" |
| Anniversary | Held on token launch anniversary | "Celebrated with the community" |
| Streak | 90 consecutive days at tier 5+ | "Consistent high conviction" |

## Additional Features

### Badge Lineage
> Premium feature tracking how badges were earned

### Custom Badges
> Create community-specific badges with manual or automatic criteria

### Discord Integration
Displayed as:
- Separate roles
- Nickname prefixes (e.g., `[Diamond][Whale] Username`)
- Channel access
