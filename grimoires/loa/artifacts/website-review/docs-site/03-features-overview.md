**URL:** https://docs.arrakis.community/features

---

## Core Features

### 1. Conviction Scoring
> Algorithmic evaluation measuring "holding duration, trading patterns, accumulation behavior, and on-chain activity" to identify committed members.

### 2. 9-Tier Progression
> Customizable member segmentation rewarding "long-term commitment over short-term speculation."

### 3. Badge System
> Achievement recognition for milestones like early adoption, market downturn survival, and dip accumulation.

### 4. Shadow Mode
> Testing capability alongside existing Collab.Land setups with result comparison before full deployment.

### 5. Analytics Dashboard
> Community composition tracking including tier distribution, conviction trends, and holder behavior patterns.
