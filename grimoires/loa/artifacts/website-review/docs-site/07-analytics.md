## Key Metrics Categories

### Holder Metrics

| Metric | Description |
|--------|-------------|
| Total Holders | Unique wallets holding tokens |
| Verified Members | Discord users who connected wallets |
| Verification Rate | Percentage of token holders verified |

### Conviction Metrics

| Metric | Description |
|--------|-------------|
| Average Conviction | Mean score across all members |
| Median Conviction | "Median score (less affected by outliers)" |
| Conviction Gini | Measures conviction distribution equality |

### Retention Metrics

| Metric | Description |
|--------|-------------|
| 30-Day Retention | "% of members maintaining tier for 30+ days" |
| Tier Stability | Average duration at current tier |
| Churn Rate | "% of members who dropped tiers" |

## Access & Privacy

Analytics data available via API for Growth/Enterprise plans only.

Individual member visibility restricted to admins with permissions; aggregated by default.

### Plan Features

| Feature | Starter | Growth | Enterprise |
|---------|---------|--------|------------|
| Basic Dashboard | Yes | Yes | Yes |
| Trend Analysis | No | Yes | Yes |
| Weekly Reports | No | Yes | Yes |
| Data Export | No | Yes | Yes |
| API Access | No | Read | Full |
| Custom Reports | No | No | Yes |
