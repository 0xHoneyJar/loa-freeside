## What is ARRAKIS?

> ARRAKIS functions as a conviction scoring and tiered progression system for token-gated communities, transforming blockchain data into actionable community insights delivered via Discord roles.

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Conviction Scoring** | Algorithmic evaluation based on holding duration, trading patterns, and on-chain activity |
| **9-Tier Progression** | Member segmentation beyond basic token-gating |
| **Badge System** | Achievement badges for community milestones |
| **Shadow Mode** | Test with existing Collab.Land setup risk-free |
| **Auto-Refresh** | Roles update every 6 hours automatically |
