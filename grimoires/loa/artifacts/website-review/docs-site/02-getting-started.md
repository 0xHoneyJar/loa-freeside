# Getting Started - docs.arrakis.community/getting-started
**URL:** https://docs.arrakis.community/getting-started

---

## Overview

> "Get ARRAKIS running in your Discord server in 15 minutes or less."

---

## Prerequisites

- A Discord server with admin permissions
- An ERC-20 or NFT token contract
- Collab.Land bot installed (optional, for Shadow Mode)

---

## Step 1: Join the ARRAKIS Discord

> Visit discord.gg/thehoneyjar to access the setup wizard.

---

## Step 2: Connect Your Server

> Authorize ARRAKIS with minimum permissions:
- Manage Roles
- Read Messages
- Send Messages

---

## Step 3: Configure Your Token

Enter token contract details:

| Field | Description |
|-------|-------------|
| Chain | Ethereum, Polygon, Arbitrum, Optimism, Base, or other EVM chains |
| Contract Address | Your token's contract address |
| Token Type | ERC-20 or ERC-721/1155 |

---

## Step 4: Set Up Tiers

Configure up to 9 tiers:

| Tier | Name | Threshold |
|------|------|-----------|
| 1 | Initiate | Any balance |
| 2 | Acolyte | 100 tokens |
| 3 | Adherent | 500 tokens |
| 4 | Disciple | 1,000 tokens |
| 5 | Zealot | 5,000 tokens |
| 6 | Elder | 10,000 tokens |
| 7 | Oracle | 25,000 tokens |
| 8 | Prophet | 50,000 tokens |
| 9 | Kwisatz Haderach | 100,000 tokens |

---

## Step 5: Enable Shadow Mode (Recommended)

> Shadow Mode allows parallel testing without affecting current roles, enabling threshold comparison before deployment.

---

## Step 6: Go Live

> Disable Shadow Mode when ready for full role management.
